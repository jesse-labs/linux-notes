# Basic Linux Commands

## Pwd
show the absolute path of the directory you're in

## ls
 lists out the directories in the path you're in 

## ls -la
lists every directory and file hidden and unhidden in the path you're in

## cd
to open a directory

## mkdir
test to create a directory called test 

## touch
file.txt to create a empty file

## cp 
file.txt file2.txt to copy file.txt contents to a new or already existing file called file2.txt

## mv 
file2.txt renamed.txt renames a file 

## rm 
renamed.txt to remove renamed.txt

## rmdir
test to remove the directory test

## cat
/etc/os-release to see the contents of a file 

## whoami 
says the usersname

## id 
tells the userid of every user aslong with their username 

## hostname 
says the hostname

## ip addr 
Shows all network interfaces and their IP addresses MAC addresses and status

## ip route 
view the routing table

## df -h 
tells the amount of free storage space in a human readable format

## free -h 
tells the amount of free ram in a human readable format

## uname -a 
displays kernel information and architecture
